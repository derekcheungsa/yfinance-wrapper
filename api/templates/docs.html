API Documentation

Rate Limiting

The API implements rate limiting of 100 requests per minute per IP address.

Authentication

No authentication required for public endpoints.

Base URL

All endpoints are relative to: /api

Get Current Stock Data

GET /stock/{ticker}

Returns current stock data including price, change, market cap, and volume.

Parameters

ticker (path, required) - Stock symbol (e.g., AAPL)

Response Example

{
    "symbol": "AAPL",
    "price": 150.23,
    "change": 2.45,
    "change_percent": 1.63,
    "volume": 82451234,
    "market_cap": 2450000000000
}

Get Historical Stock Data

GET /stock/{ticker}/history

Returns historical price data for a given stock.

Parameters

ticker (path, required) - Stock symbol
period (query, optional) - Time period (1d, 5d, 1mo, 3mo, 6mo, 1y, 2y, 5y, 10y, ytd, max)
interval (query, optional) - Data interval (1m, 2m, 5m, 15m, 30m, 60m, 90m, 1h, 1d, 5d, 1wk, 1mo, 3mo)

Get Company Information

GET /stock/{ticker}/info

Returns detailed company information and profile.

Parameters

ticker (path, required) - Stock symbol

Get Moving Averages

GET /stock/{ticker}/analysis/moving-averages

Returns Simple Moving Average (SMA) and Exponential Moving Average (EMA) calculations.

Parameters

ticker (path, required) - Stock symbol
period (query, optional) - Period for calculation (default: 20)

Get RSI Analysis

GET /stock/{ticker}/analysis/rsi

Returns Relative Strength Index (RSI) calculation.

Parameters

ticker (path, required) - Stock symbol
period (query, optional) - Period for calculation (default: 14)

Get Statistical Analysis

GET /stock/{ticker}/analysis/statistics

Returns statistical analysis including volatility and returns.

Parameters

ticker (path, required) - Stock symbol

Get Options Chain Data

GET /stock/{ticker}/options

Returns options chain data including calls, puts, and implied volatility.

Parameters

ticker (path, required) - Stock symbol
expiration_date (query, optional) - Options expiration date (YYYY-MM-DD)
