Get Stock Data

GET /stock/{ticker}

Returns current stock data including price, change, market cap, and volume.

Parameters

ticker (path, required) - Stock symbol (e.g., AAPL)

Response Example

{
    "symbol": "AAPL",
    "price": 150.23,
    "change": 2.45,
    "change_percent": 1.63,
    "volume": 82451234,
    "market_cap": 2450000000000
}

Error Responses

  • 404 - Invalid ticker symbol
  • 429 - Rate limit exceeded
